# Homebrew Tap for Milaidy

Personal AI assistant built on [ElizaOS](https://github.com/elizaos) â€” cute agents for the acceleration.

## Install

```bash
brew tap milady-ai/tap
brew install milaidy
```

Or one-liner:

```bash
brew install milady-ai/tap/milaidy
```

## Usage

```bash
milaidy start          # Start the agent runtime
milaidy setup          # Initialize workspace
milaidy dashboard      # Open the web dashboard
milaidy --help         # Show all commands
```

## Requirements

- **Node.js** >= 22 (installed automatically via Homebrew dependency)
